# Introdução à Linguagem Go

Bem-vindo à documentação oficial do **Time 7** sobre a linguagem Go (Golang). Go é uma linguagem de programação de código aberto projetada pelo Google para ser simples, eficiente e confiável.

## Por que aprender Go?
Go combina a performance de linguagens compiladas (como C++) com a facilidade de desenvolvimento de linguagens dinâmicas (como Python).

!!! info "Destaque da Linguagem"
    Go foi criada para resolver problemas de escalabilidade e concorrência, tornando-a ideal para sistemas distribuídos e infraestrutura de nuvem.

---

## Instalação passo a passo

Para começar a desenvolver, siga as etapas abaixo:

1. Acesse o site oficial: [golang.org/dl](https://golang.org/dl/)
2. Baixe o instalador correspondente ao seu sistema operacional (Windows, macOS ou Linux).
3. Após a instalação, verifique se tudo está correto abrindo seu terminal e digitando:

```bash 
go version
```

!!! warning "Configuração de Ambiente"
Certifique-se de que o binário do Go foi adicionado ao seu PATH. Caso contrário, o comando go não será reconhecido pelo terminal.

## Primeiros Passos
Para criar seu primeiro programa, utilize o comando de inicialização de módulos:

```bash 
go mod init hello-world
```
